# 🌿 فرص خضراء | Green Opportunities

منصة ويب تعرض المنح والتدريبات والمسابقات البيئية المخصصة لشباب الوطن العربي، مع إمكانية البحث والتصفية بسهولة.

A web platform showcasing green grants, training programs, and competitions for Arab youth — with built-in search and filtering.

---

## ✨ المميزات | Features

- 🔍 **بحث فوري** بعنوان الفرصة
- 🎯 **تصفية حسب النوع**: منحة / تدريب / مسابقة
- 💰 **تصفية حسب حالة التمويل**: ممولة بالكامل / جزئياً / غير ممولة
- 🔄 **إعادة ضبط** سريعة لكل الفلاتر
- 📱 **تصميم متجاوب** (Responsive) يعمل على جميع الأجهزة
- 🌐 **دعم كامل للغة العربية** (RTL)
- 🎨 واجهة عصرية بألوان مستوحاة من الطبيعة (أخضر وبرتقالي)

---

## 🗂️ هيكل المشروع | Project Structure

```
├── index.html      # الصفحة الرئيسية
├── style.css       # التنسيقات والتصميم
├── script.js       # منطق البحث، التصفية، وعرض البطاقات
└── images/
    └── original.png   # شعار المنصة
```

## 🛠️ التقنيات المستخدمة | Tech Stack

| التقنية | الاستخدام |
|---|---|
| **HTML5** | بنية الصفحة |
| **CSS3** (Custom Properties, Flexbox, Grid) | التصميم والتجاوب |
| **JavaScript (Vanilla)** | منطق التصفية والبحث الديناميكي |
| **Font Awesome** | الأيقونات |

لا توجد أي تبعيات (dependencies) أو أدوات بناء — المشروع يعمل مباشرة في المتصفح.

## 🚀 طريقة التشغيل | Getting Started

1. قم بتنزيل أو استنساخ المشروع:
   ```bash
   git clone <repo-url>
   ```
2. افتح ملف `index.html` مباشرة في المتصفح، أو استخدم خادم محلي بسيط:
   ```bash
   npx serve .
   ```
3. استمتع بتصفح الفرص الخضراء! 🌱

## 📊 مصدر البيانات | Data Source

البيانات حالياً تجريبية (Mock Data) ومخزنة داخل `script.js` كمصفوفة من الكائنات، كل كائن يحتوي على:

```js
{
  id: 1,
  title: "عنوان الفرصة",
  type: "منحة | تدريب | مسابقة",
  country: "الدولة",
  deadline: "YYYY-MM-DD",
  funding: "ممولة بالكامل | ممولة جزئياً | غير ممولة"
}
```

> يمكن لاحقاً ربط المشروع بقاعدة بيانات أو API حقيقي بدلاً من البيانات الثابتة.
